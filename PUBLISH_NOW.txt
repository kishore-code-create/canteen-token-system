SUPPORT & HELP
═══════════════════════════════════════════════════════════════════════════════════

If apps don't start:
  1. Check the "Run" logs on Replit
  2. Look for error messages
  3. Click "Run" again

If you can't access the URLs:
  1. Make sure Replit is showing "Always On"
  2. Check Replit Settings > Multiplayer > Public (make it public)
  3. Wait 30 seconds for URL to be active

If students don't show:
  1. The database auto-initializes when apps start
  2. If empty, contact me to reload the database
  3. Check the database file exists in Replit
